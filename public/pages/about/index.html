Who I am#

Hi 👋 !

I’m TRNEEDANAME, a french game dev / mod maker. This site is my blog, where you’ll find what I did and what I’m doing.

What I use#

I use Linux as my main OS (NixOS to be precise), before that I used Arch for close to 4 years.

As my text editor / IDE / everything else (mostly), I use Emacs because I write in org-mode (I use a framework called Doom Emacs).

The file you’re reading is in fact a .org file !

What I do#

I make mods for Project Zomboid and Xcom in my spare time, I started with Xcom 2 in ~2020, in 2024 I joined the Project Zomboid RP scene as a player, then admin / mod maker for another server.

I also play games, as of now I’m playing Satisfactory / Project Zomboid.

I enjoy writing wikis, as show by my Militia wiki for Xcom 2 (still in the work as of now)

Technology used by this site#

This site is made using Hugo with ox-hugo, I have Hugo admonitions as a plugin installed, but as it doesn’t work with org-mode, I had to write a minor mode for it.
